# TicTacToe Game

Juego de TicTacToe construido con React, TypeScript y Tailwind CSS.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Características

- Diseño responsivo
- Sistema de puntuación
- Efectos de sonido
- Interfaz moderna
